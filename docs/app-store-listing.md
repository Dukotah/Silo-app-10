## Developer Notes

- Replace all `your-website.com` placeholders with actual URLs before submission.
- Replace `your@email.com` in `privacy-policy.html` with your real contact email.
- Replace `REVENUECAT_IOS_KEY` / `REVENUECAT_ANDROID_KEY` in `src/useVIP.js` with your real RevenueCat API keys.
- Replace `com.yourname.silo` in `capacitor.config.ts` with your actual bundle ID.
- Run `npx @capacitor/assets generate` (after installing `@capacitor/assets`) to generate all icon/splash sizes from `public/icon.svg` and `public/splash.svg`.
- Run `npm run mobile:ios` (requires macOS + Xcode) to open the iOS project.
- Run `npm run mobile:android` (requires Android Studio) to open the Android project.
